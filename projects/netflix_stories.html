Netflix Stories



About

Netflix Stories is an interactive narrative game delivering branching storylines and dynamic player choices. Players can shape the story through actions, dialogue, and interactions, supported by responsive UI, immersive camera and environment systems, and scalable content pipelines.


My Contributions

As a Senior Software Engineer, I led the development of many player-facing systems, including:

  • Character creation and customization tools, enabling players to craft unique avatars.
  • Minigames, interactive moments, and branching narrative triggers.
  • Menus, HUDs, camera, and environmental rulesets that adapt dynamically to story and player choices.

I also built robust internal tools for designers:

  • Logic-graph system for defining complex narrative flows, actions, and environment changes.
  • Proto data generation pipelines from designer specifications, ensuring consistent and scalable content delivery.

Beyond feature development, I:

  • Mentored junior engineers and set standards for scalable, maintainable UI and gameplay systems.
  • Designed technical solutions for high-performance, data-driven systems under tight production constraints.
  • Supported live operations post-launch, iterating on features to improve engagement and stability.
